# SQL Sales Analysis Project

## Project Overview
This project analyzes sales performance using SQL. The dataset contains sales transactions, product details, and customer information. The goal is to generate business insights using SQL queries.

## Dataset
The project uses the following dataset:

- sales_data.csv

Columns include:
- Order_ID
- Order_Date
- Customer_ID
- Product_ID
- Units_Sold
- Sales_Rep
- Unit_Price
- Revenue

## SQL Concepts Used
The analysis demonstrates the following SQL skills:

- SELECT
- Aggregations (SUM)
- GROUP BY
- ORDER BY
- JOIN operations
- Business analysis queries

## Example Business Questions Answered

1. Total revenue generated
2. Total units sold
3. Revenue by product category
4. Revenue by customer type
5. Top products by revenue
6. Revenue by category and product

## Files in this Repository

- sales_analysis.sql → contains all SQL queries used for analysis  
- sales_data.csv → dataset used for analysis
